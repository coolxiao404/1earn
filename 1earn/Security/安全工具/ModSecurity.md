**简介**

ModSecurity 是一款开源的跨平台 Web 应用程序防火墙（WAF）引擎，用于 Apache，IIS 和 Nginx，由 Trustwave 的 SpiderLabs 开发。作为 WAF 产品，ModSecurity 专门关注 HTTP 流量，当发出 HTTP 请求时，ModSecurity 检查请求的所有部分，如果请求是恶意的，它会被阻止和记录。
